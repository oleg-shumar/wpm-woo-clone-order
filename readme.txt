This plugin adds One-Click Clone feature for each Woo Order at Orders List

== Description ==
* Author: [WP Masters](https://wp-masters.com)
* Project URI: <https://wp-masters.com/woo-clone-order>

This plugin is really helpful when you need to create a new order for existing customer or when you get orders by phone.
After activation you will find a new Clone button in WP Admin > Woo > Orders list page within the order actions.
Click on that button will create a new order copy of chosen one.
You will be redirected to cloned order with customer data and products inside.
If you don't see the Actions column on the right side of Woo Orders list, click Screen Options at the top right of the page.

= Tags =
Woo, clone orders, clone order, duplicate order, copy order, clone, duplicate

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly;
2. Activate the plugin through the 'Plugins' screen in WordPress
3. There will now be a Clone button in the Woo orders list (Woo > Orders)
4. If you don't see the Action column on the right side of Woo orders list, click Screen Options at the top right of the page

= How to use this plugin? =
Simply click the Clone button and you will be redirected to new cloned order.
